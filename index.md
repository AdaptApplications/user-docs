# Adapt Apps User Documentation

Welcome to the official documentation for the **Adapt Apps User Console**. This guide is designed to help you navigate and utilise the various features of the console effectively.

## Getting Started

If you're new to the Adapt Apps User Console, we recommend starting with the [User Console overview](docs/user_console/index.md).

## Key Features

### [Dashboard](docs/user_console/dashboard/index.md)
Get a high-level view of your activities and system status. The dashboard provides real-time insights and quick access to essential tools.

### [Item Management](docs/user_console/item/index.md)
Learn how to create, view, and manage items within the system. This section covers:
- [Creating Items](docs/user_console/item/create_item/index.md) (Normal, Excel Import, Public, and System)
- [Viewing Items](docs/user_console/item/viewing_items/index.md)
- [Item Actions](docs/user_console/item/item_actions/index.md)

### [Notifications](docs/user_console/notifications/index.md)
Stay informed with system alerts and updates.
- [Creating Alerts](docs/user_console/notifications/create_alert/create_alert.md)
- [Viewing Notifications](docs/user_console/notifications/viewing_notifications/viewing_notifications.md)

### [Reports](docs/user_console/reports/index.md)
Generate and analyse reports to gain deeper insights into your data and operations.

### [User Management](docs/user_console/user_management/index.md)
Manage users and teams within your organisation.
- [Creating Users](docs/user_console/user_management/create_user/create_user.md)
- [Deleting Users](docs/user_console/user_management/delete_user/user_delete.md)
- [Managing Teams](docs/user_console/user_management/teams/manage_team.md)
